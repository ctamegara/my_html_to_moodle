Let \(X\) be a random variable. We know that
  • \(P(X\in [0,2] )= .7 \)
  • \(P(X\in [1,3] )= .7\)
Prove that \(P(X\in[1,2])\geq .4\). We note that, writing $I=[0,2]$ and $J=[1,3]$, we have $I\cap J=[1,2]$. From the formula $$ P(X\in I\cup J)=P(X\in I)+P(X\in J)-P(X\in I\cap J)$$ we deduce the equality $$ P(X\in I\cap J)]=P(X\in I)+P(X\in J)-P(X\in I\cup J) \ .$$ Moreover, we know $P(X\in \text{{whatever}})\leq 1$, so $$P(X\in I\cup J)\leq 1\ .$$ Finally : $$ P(X\in I\cap J)]=.7+.7-P(X\in I\cup J)\geq 1.4-1=.4$$ Find the error in the following code: def f(x): for i in range(100): if x = i : print(x) The operator "=" affects a value to a variable. Here we should use the comparison operator "==": def f(x): for i in range(100): if x == i : print(x)
